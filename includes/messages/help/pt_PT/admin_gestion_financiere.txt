<h1>Gest&atilde;o financeira dos leitores</h1>
<p>A gest&atilde;o financeira dos leitores permite gerar os
movimentos financeiros associados a cada leitor : 
</p>
<ul>
	<li>empr&eacute;stimos 
	<li>multas como consequ&ecirc;ncia da
	devolu&ccedil;&atilde;o atrasada dos documentos 
	<li>tarifa dos empr&eacute;stimos segundo o tipo de documento 
	
</ul>
<p>Se escolheu a gest&atilde;o simples das tarifas das inscrições,
estas est&atilde;o vis&iacute;veis em <a href='admin.php?categ=empr&sub=categ'>Administra&ccedil;&atilde;o;
Leitores; Categor&iacute;as de leitores</a>.</p>
<p>Se escolheu a gest&atilde;o simples das tarifas dos empr&eacute;stimos,
estas est&atilde;o vis&iacute;veis em <a href='admin.php?categ=docs&sub=typdoc'>Administrac&atilde;o
; Exemplares ; Tipos de documentos</a>.</p>
