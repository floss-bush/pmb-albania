<h1>Administration des notices</h1>

<p>La section <b>administration des notices</b> permet de d&eacute;finir les param&egrave;tres de l'application pour les notices.
<ul>
<li><b>origines</b> permet de d&eacute;finir les origines des notices, afin de distinguer les droits de rediffusion</li>
<li><b>statuts</b> le statut des notices permet de faire des distinctions de traitement pour les notices, ainsi que dans l'affichage dans l'OPAC.</li>
<li><b>personnalisables</b> champs personnalisables sur les notices</li>
</ul></p>
