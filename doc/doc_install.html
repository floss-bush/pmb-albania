Installation de la documentation en ligne dans PMB

Téléchargez la documentation depuis www.sigb.net
documentation.zip

Décompressez documentation.zip dans VOTRE_REP_PMB/doc/documentation

Activer l'affichage de la documentation dans PMB
Dans Administration > Outils > Paramètres, passez la valeur du paramètre pmb show_help à 1.

Désactiver l'affichage de la documentation dans PMB
Dans Administration > Outils > Paramètres, passez la valeur du paramètre pmb show_help à 0.
